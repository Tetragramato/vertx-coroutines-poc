# Vertx-coroutine-poc

[![Vertx](https://img.shields.io/badge/vert.x-3.8.3-purple.svg)](https://vertx.io)

Vertx sample with Kotlin Coroutines.

## Build & run

To build :

```bash
./gradlew clean build
```

To run :

```bash
java -jar build/libs/vertx-coroutines-poc-1.0.0-SNAPSHOT-fat.jar
```
